## Overview

This project is an end-to-end retail analytics solution designed to extract business insights from transactional data and build a product recommendation system.

It demonstrates how data analysis and machine learning can support decision-making in e-commerce environments by improving customer understanding, product performance tracking, and personalized recommendations.

---

## Business Problem

E-commerce businesses face challenges such as:
- Understanding customer purchasing behavior
- Identifying high-value customer segments
- Improving product discovery
- Increasing cross-selling opportunities

This project addresses these problems using data analytics and a recommendation system approach.

## Recommendation System

A collaborative filtering-based recommendation system was developed using customer purchase behavior.

### Approach:
- Customer-product interaction matrix is created
- Similar customers are identified using Nearest Neighbors
- Products purchased by similar users are recommended
